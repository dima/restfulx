There‘s a number of things that ActiveRecord/ActiveSupport and the rest of the family get ~wrong~ from the point of view of Flex clients.

Some of these things are:

  • XML format (Flex really doesn‘t like dashes in XML messages)
  • Errors (we need to be more specific and communicate what went wrong where)

This is where we try to fix this stuff.

Some of the things that are done can be called monkey patching while others can be called extensions. Caveat emptor.
